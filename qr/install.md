Installation
============

QR does not have any dependencies.
It can be installed via following command:

```bash
docker run \
-p 80:80/tcp \
-d images.perfumerlabs.com/dist/qr:v1.0.0
```

Refer to [configuration page](/images/qr/config) for parameters description.
